                    myMean <- function(assignment2) {
                    return(sum(assignment) / length(someData))
                    }
Running this given function resulted in an error message returning "Error in myMean(assignment2) : object 'assignment' not found". This is due to the fact that the object assignment along with someData was not defined before trying to run sum and length function. To fix this simply convert these two pointers to assignment2 and the code will execute correctly returning a mean of 19.25. The fully corrected code block would look something like this: 

                        assignment2 <- c(16, 18, 14, 22, 27, 17, 19, 17, 17, 22, 20, 22)

                        myMean <- function(assignment2) {
                        return(sum(assignment2) / length(assignment2))
                        }

                        result <- myMean(assignment2)
                        print(result)
With Result printing 19.25
